## Overview

Khaki Gemstone is a full-stack web platform designed to modernize the gemstone industry by integrating e-commerce, investment capabilities, and service-based interactions into a single ecosystem.

The platform is structured into multiple portals, each serving a specific type of user while maintaining a unified backend system.

---

## Platform Modules

### Public Portal

The public-facing interface allows users to browse and purchase gemstones.

**Features:**
- Browse products across multiple categories (cut stones, rough stones, rings, beads, specimens)
- View detailed product information
- Add items to cart and initiate purchases
- Clean and responsive interface
- Discount and featured product handling

---

### Investor Portal

The investor portal introduces a financial layer to the platform.

**Features:**
- Invest in gemstone inventory
- Track returns and portfolio performance
- Monitor investment history
- View financial insights

---

### Admin Panel

The admin panel provides centralized control over the system.

**Features:**
- Product management (create, update, delete)
- Category and filter management
- User management
- Order and transaction tracking
- Platform monitoring and analytics

---

### Agent Hub

A service-based module designed for real-time agent interaction, particularly for car dealership workflows.

**Features:**
- "Call Closest Agent" functionality
- Location-based agent matching
- Distance and ETA calculation
- Map-based visualization
- Direct interaction between users and agents
